# Checklist photo contrôle qualité — 5 angles standards
> À réaliser à chaque rotation (après ménage, avant check-in). Envoyer les 5 photos à Vincent via WhatsApp.

## Les 5 angles obligatoires

### 1. Vue d'entrée / façade (depuis le portail ou la porte d'entrée)
- [ ] Porte/portail propre, sans traces
- [ ] Pas de déchets visibles
- [ ] Éclairage fonctionnel (la nuit)

### 2. Salon / espace principal (angle large depuis l'entrée)
- [ ] Coussin centrés, coussins bien placés
- [ ] Pas de poussière visible sur les meubles apparents
- [ ] Lumière naturelle ou éclairage allumé

### 3. Chambre principale (lit centré dans le cadre)
- [ ] Literie impeccable — pas de pli, taies droites
- [ ] Oreillers symétriques
- [ ] Sol propre, rien qui traîne

### 4. Salle de bain (miroir + vasque visibles)
- [ ] Miroir sans traces
- [ ] Vasque et robinetterie brillantes
- [ ] Serviettes pliées et disposées proprement
- [ ] Sol sec

### 5. Cuisine / coin repas (plan de travail + évier)
- [ ] Plan de travail vide et propre
- [ ] Évier propre et sec
- [ ] Électroménager rangé

## Format d'envoi
`[Bien] [Date] [Heure] — OK ✅` ou `[Bien] [Date] — ⚠️ [problème]` + photo du problème
