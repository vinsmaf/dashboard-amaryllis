## 🔄 Rotation standard (ménage entre deux séjours)

### Timing cible
- **Départ** : libération du logement à 11h max
- **Arrivée** : logement prêt à 16h (fenêtre de 5h)
- **Alerte anticipée** : si rotation < 3h (départ 13h / arrivée 16h), Vincent prévient le prestataire J-2

### Checklist rotation complète

**Accueil & sécurité**
- [ ] Récupérer les clés ou vérifier la serrure à code (code = changé ?)
- [ ] Inventaire rapide : mobilier intact, aucun dégât visible
- [ ] Vérifier que le logement est bien libéré (pas d'oubli de bagages)

**Nettoyage (ordre recommandé)**
- [ ] Ouvrir toutes les fenêtres (aération 30 min minimum)
- [ ] Vider toutes les poubelles + remplacer les sacs
- [ ] Vaisselle / lave-vaisselle : lancer si plein, vider si terminé
- [ ] Nettoyage cuisine : plans de travail, plaques, four, réfrigérateur intérieur
- [ ] Salle(s) de bain : WC, lavabo, douche/baignoire, miroirs
- [ ] Sols : aspiration puis lavage (toutes pièces)
- [ ] Mobilier : dépoussiérage surfaces, coussins retournés
- [ ] Terrasse/balcon : balayage, essuyage mobilier extérieur
- [ ] Piscine/jacuzzi : vérification visuelle propreté + niveau eau (→ signaler si anormal)

**Linge**
- [ ] Retrait complet du linge utilisé (draps, serviettes, torchons)
- [ ] Mise en machine si laissée sur place, sinon sac pour blanchisserie
- [ ] Installation du linge propre (draps tendus, pliage soigné)
- [ ] Serviettes : 2 par personne (bain + visage), disposées proprement
- [ ] Torchons cuisine : 2 minimum

**Dotations**
- [ ] Savon / gel douche / shampoing (refaire si < moitié)
- [ ] Papier WC (minimum 4 rouleaux)
- [ ] Éponge vaisselle et produit vaisselle
- [ ] Sacs poubelle (minimum 5)
- [ ] Café/thé/sucre selon la dotation de la propriété

**Contrôle final**
- [ ] Fenêtres fermées, volets selon consigne
- [ ] Climatisation éteinte ou réglée à 26°C (consigne propriété)
- [ ] Toutes les lumières éteintes
- [ ] Code d'accès : vérifier que le bon code est configuré pour les prochains arrivants
- [ ] Photos de contrôle (3 minimum : salon, chambre principale, cuisine)

## 🔧 Intervention maintenance

### Niveaux d'urgence

| Niveau | Délai d'intervention | Exemples |
|---|---|---|
| 🔴 **Urgence** | < 2h | Clim hors service (canicule), fuite eau, serrure bloquée, panne électrique totale |
| 🟠 **Important** | < 24h | Clim sous-performante, chauffe-eau défaillant, équipement cassé critique |
| 🟡 **Normal** | 3–7 jours | Ampoule grillée, équipement non critique défaillant, petite réparation |
| ⚪ **Planifié** | Prochain créneau inter-séjour | Peinture, entretien préventif, remplacement consommables |

### Protocole intervention

1. **Signalement** → Vincent reçoit l'alerte (voyageur ou prestataire)
2. **Diagnostic à distance** → WhatsApp photo/vidéo en 15 min
3. **Décision** → Vincent choisit le prestataire et donne le feu vert
4. **Accès** → Code communiqué au technicien (jamais de clé laissée)
5. **Rapport** → Photo avant/après dans le groupe WhatsApp prestataire
6. **Facturation** → Devis validé avant toute intervention > 100€

## 📸 Photos de contrôle — quand et comment

**Après chaque rotation** : 3 photos minimum (salon, chambre, cuisine)  
**Après chaque incident** : avant/après obligatoire  
**Avant remise des clés** : 1 photo de chaque pièce  

**Stockage** : WhatsApp groupe propriété + dossier iCloud par bien et par date  
**Format** : `YYYY-MM-DD_bien_rotation.jpg` ou `YYYY-MM-DD_bien_incident_[description].jpg`
