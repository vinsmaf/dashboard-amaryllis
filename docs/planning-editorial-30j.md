# Planning éditorial Amaryllis Locations — 30 jours

> **Période couverte** : 27 mai → 25 juin 2026 (inter-saison Martinique + Nogent stable)
> **Plateformes** : Facebook (Page) + Instagram (PBIA — page-backed)
> **Cible** : 30 publications réparties sur 7 biens, jamais 2× le même angle d'affilée

## Voice & règles canon

- **"Vous"** formel toujours
- Ton : sensoriel, premium, jamais publicitaire
- Mots interdits : *opportunité unique, ne ratez pas, n'hésitez pas, contactez-nous*
- CTAs canoniques rotatifs : `Réserver` · `Découvrir` · `Sauvegarder` · `Voir la visite` · `Tag a friend`

## Saisonnalité Mai-Juin 2026

- **Inter-saison Martinique** : Mai-Juin = peu de touristes français, prix doux, faune luxuriante, soleil sûr, pas de cyclones
- **Avantage à mettre en avant** : *"l'île pour vous seuls"*, *"prix d'inter-saison sans compromis"*, *"saison verte = jardins luxuriants"*
- **Nogent** : indépendant — promouvoir week-ends parisiens + bord de Marne au printemps

## Structure hebdomadaire (7 jours-types)

Chaque jour de la semaine a un **thème principal** + **4 variantes** rotatives sur 4 semaines (anti-lassitude).

### Lundi — Inspiration / projection
- Variante 1 : *"Imaginez votre réveil…"* (sensoriel matin)
- Variante 2 : *"Six heures du matin face à la mer"* (instant figé)
- Variante 3 : *"Lever de soleil dans une chambre baignée de lumière"*
- Variante 4 : *"Vue depuis la terrasse à l'aube"*
- **Format prioritaire** : Reel 15-30s vue mer/jardin au lever
- **CTA** : Voir la visite

### Mardi — Preuve sociale
- V1 : Citation d'avis 5★ extraite + photo correspondante
- V2 : *"X voyageurs nous ont laissé 4,X étoiles"* + carrousel ambiance
- V3 : Avant/après transformation (saison sèche vs verte)
- V4 : Témoignage anonyme storyfié (1 paragraphe court)
- **Format** : Carrousel 5-7 photos OU post avec citation overlay
- **CTA** : Voir la visite

### Mercredi — Détail premium
- V1 : Focus piscine (eau, lumière, son)
- V2 : Focus terrasse / coin lecture / vue panoramique
- V3 : Focus suite parentale ou cuisine équipée
- V4 : Focus équipement signature (jacuzzi, BBQ, hammac)
- **Format** : Reel ambiance (eau, mouvement, lumière) — son d'ambiance, pas de musique
- **CTA** : Découvrir

### Jeudi — Rêve émotionnel / storytelling
- V1 : *"Un week-end parfait à [Bien]"* (récit 24h)
- V2 : *"Le rituel du matin chez [Bien]"* (petit-déj face mer)
- V3 : *"Quand la nuit tombe à [Bien]"* (jacuzzi + étoiles)
- V4 : *"3 instants à ne pas manquer à [Bien]"* (carrousel narratif)
- **Format** : Reel immersif 30-60s OU carrousel 5 photos narratives
- **CTA** : Réserver

### Vendredi — Conversion / offre / dispo
- V1 : Disponibilités weekend prochain (sans urgence factice)
- V2 : Offre inter-saison : 4 nuits payées, 5ème offerte
- V3 : Pack 3-jours : villa + transfert aéroport + 1 dîner
- V4 : Mise en avant tarif transparent vs OTA (-10% direct)
- **Format** : Reel dynamique check-in OU carte tarifs claire
- **CTA** : Réserver maintenant

### Samedi — Lifestyle destination
- V1 : *"5 plages secrètes autour de [Bien]"*
- V2 : *"Le marché local du samedi"* (épices, rhum, fleurs)
- V3 : *"Restaurants signature à 15 min de [Bien]"*
- V4 : *"Plongée / kitesurf / catamaran à proximité"*
- **Format** : Carrousel 5-7 photos lifestyle + plan d'accès
- **CTA** : Sauvegarder pour plus tard

### Dimanche — Information / projection long-terme
- V1 : Comparatif inter-saison vs haute saison (calme + prix)
- V2 : *"Que mettre dans la valise pour Sainte-Luce ?"*
- V3 : *"Quand partir en Martinique selon votre profil"* (couple/famille/groupe)
- V4 : *"Comment se déplacer en Martinique"* (voiture / scooter / taxi)
- **Format** : Post statique très esthétique + texte info
- **CTA** : Voir la visite

## Workflow opérationnel

```
J-2 (06h Martinique)
  → Cron Worker → /api/agents-run avec calendar_id du jour J
  → community-manager génère le draft selon brief calendrier (bien, thème, variante, format)
  → validator (panel multi-agent) reformat en 5 blocs
  → draft.status = pending (visible dans onglet Approbations)

J-1 (soir)
  → Tu reviews dans Approbations
  → ✅ Approve OU ✏️ Édite OU 🚫 Reject (et regen)

J jour-J (H-15 min)
  → Cron Worker checke les drafts approved scheduled_for ≤ now
  → /api/social publish → Facebook + Instagram (PBIA)
  → draft.status = published, calendar entry → published_at = now
```
